## 它到底能干什么？

拿几个实际场景说：

**多个项目同时跑，版本还各不一样**

这是很多 Mac 开发者的日常。A 项目用 PHP 7.4，B 项目要 PHP 8.3，C 项目还依赖老版 MySQL 5.7。用 Homebrew 手动管这些，光是切版本就够折腾的，而且很容易搞乱系统环境。

ServBay 支持给每个项目独立设定语言版本和数据库，项目之间完全隔离，互不干扰。一台机器上同时跑六七个项目，也不会有"环境污染"的问题。

**本地开发用真实域名 + SSL 证书**

很多工具做不到这一点，但 ServBay 内置了 DNS 服务，可以使用未注册的自定义域名，甚至非标准 TLD（比如 `.local`、`.test`）。Pro 版还支持通过 ACME 协议自动申请 Let's Encrypt 正式证书，并自动续签，开发时浏览器也能看到绿锁。

**内置邮件服务器**

做注册、密码重置、订单通知这些功能时，以前要么连真实邮箱测试，要么配一堆 SMTP 参数。ServBay Pro 内置了邮件服务器，零配置支持 POP3/SMTP + SSL/TLS，本地直接测邮件发送逻辑。

**一键跑本地 AI 大模型**

没错，ServBay 集成了 Ollama，支持在本地部署和运行大语言模型。你可以用它跑 DeepSeek、Qwen 这类开源模型，甚至用闲置的 Mac Mini 搭一个家庭 AI 开发服务器——这已经有不少用户在实践了。
